# turboquant-experiments

Open comparative benchmarks of TurboQuant applied to two autoregressive transformer models:

1. **language-model-improvements/** — TurboQuant on Llama-3.1-8B-Instruct (text generation).
2. **speech-tts-improvements/** — TurboQuant on Microsoft VibeVoice-1.5B (autoregressive TTS).
